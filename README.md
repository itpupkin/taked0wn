### Installation

1. Клонируйте репозиторий

```sh
git clone https://github.com/itpupkin/taked0wn.git
```

2. Перейдите в директорию

```sh
cd taked0wn
```

3. Установите требуемые пакеты

```sh
pip install -r requirements.txt
```

4. Запустите

```sh
python taked0wn.py
```
